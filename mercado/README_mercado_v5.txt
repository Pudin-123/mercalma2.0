mercado_v5_carrito_pago_ready - README\n-------------------------------------\n\nContenido: proyecto Django con carrito (guest+user), modelos Order ready for Stripe and MercadoPago.\n\nPasos para probar localmente (Django 5.x):\n\n1) Extraer y entrar al directorio que contiene manage.py\n2) Instalar dependencias (ej: pip install -r requirements.txt)\n   - Para Stripe: pip install stripe\n   - Para MercadoPago: pip install mercadopago\n3) Crear migraciones y migrar:\n   python manage.py makemigrations\n   python manage.py migrate\n4) Crear superuser (opcional) y levantar server:\n   python manage.py createsuperuser\n   python manage.py runserver\n5) El panel de productos está en /market/ ; carrito en /market/cart/\n\nIMPORTANTE:\n- Añadí placeholders en config/settings.py para claves STRIPE_* y MERCADOPAGO_ACCESS_TOKEN.\n- No subas tus claves a repositorios públicos.\n
